Cosas a tener en cuenta: Los input no tienen .innerText, en vez de eso, usan .value. Mozilla.org/input

Los demás elementos usan .innerText para acceder y modificar al texto que aparece dentro. Mozilla.org/HTMLElement También pueden usar .textContent, las diferencias no son importantes por ahora.

Para evitar que el formulario form se “mande” y por ende recargue la página, al event handler “onclick”, agréguentle un return false; al final de la función. Ejemplo: const $botonCalcular = document.querySelector("#boton-calcular"); $botonCalcular.onclick = function() código que se ejecuta cuando le hagan click a este elemento... return false;
